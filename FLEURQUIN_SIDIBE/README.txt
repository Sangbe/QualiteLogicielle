	-----------------------------------------------------
			LANCEMENT DU JEU :
	----------------------------------------------------
	
	1 - Décompresser l'archive FLEURQUIN_SIDIBE.zip
	2 - Ouvrir un terminal
	3 - Se rendre dans le repertoire FLEURQUIN_SIDIBE
	4 - Taper : java -jar exploration.jar
	5 - Exécuter, et amusez-vous.


	--------------------------------------------------------
			LES COMMANDES DU JEU : 
	--------------------------------------------------------

 * look : Cette commande permet d’afficher la description d’un objet ou d’une pièce.	 


 * go : Cette commande permet de se déplacer dans le jeu, allé d’un lieu à un autre, 
	à condition qu’il existe une sortie entre ces deux lieux.	


 * use : Permet d’utiliser un objet de son inventaire. Toutefois, certains objets ne s’utilisent 
	 que sur des cibles tandis que d’autres non.
	
	 Lors d’un combat, il tout à fait possible d’utiliser une arme pour se défendre:
	


 * take : Permet de ramasser un objet dans la pièce dans laquelle on se trouve pour la mettre dans son inventaire.
	 

 * attack : Permet de se défendre à mains nus lors d’un combat. 
	    Inutile de spécifier l’adversaire, car cette commande n’est valide que lors des combats.

 
 * inventory : Permet d’afficher la liste des objets dont le héros dispose ainsi que leurs quantités.

 
 * map : Permet d’afficher la carte du jeu.


 * help : Permet d’afficher l’aide.


 * quit : Permet de quitter la partie en cours.
